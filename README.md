# Linked-Lists
Singly Linked List Project

# Features 
- ##append() adds a new node to the end of the list
- ##prepend() adds a new node to the start of the list
- ##size() returns the total number of nodes in the list
- ##at(index) returns the node at the given index
- ##pop() removes the last node from the list
- ##contains(value) returns true if list contains a node with the given value, otherwise returns false
- ##find(value) returns the index of the node with the given value
- ##toString() returns a string structured from the list objects in the format (value) -> (value) -> (value) -> null
- ##insertAt(value, index) inserts a new node with the provided value at the given index
- ##removeAt(index) removes the node at the given index
